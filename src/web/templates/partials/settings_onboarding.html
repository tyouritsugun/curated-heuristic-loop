First-time checklist

Follow these steps in order; this page keeps the secrets on disk and stores only metadata in SQLite.

  1. Prepare scripts_config.yaml. Copy scripts/scripts_config.yaml.sample, set data_path, google_credentials_path, and the sheet IDs/worksheet names.
  2. Load & verify. Use the “Load & Verify” button to register the YAML. The UI reads metadata only; it never edits your config file.
  3. Select models. Pick the embedding and reranker repos/quantization that match the hardware you have available. The worker pool will reuse these settings automatically.
  4. Validate connectivity. Use the diagnostics panel to re-check credentials + sheet access, then download a JSON backup.
  5. Head to Operations. Once every section shows a green status pill, open the Operations dashboard to run imports, exports, or index rebuilds.
