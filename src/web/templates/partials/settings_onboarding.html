Settings & Diagnostics

This page displays system status and provides diagnostic tools. Configuration is managed via .env file.

New User Setup

If you haven't set up the system yet, follow these steps:

  1. Copy .env.sample to .env and fill in your credentials and sheet IDs
  2. Run python scripts/setup.py to initialize the database and download models
  3. Start the server and begin using the Operations dashboard

See README.md for detailed setup instructions.

What's on this page

  • Configuration Status: View current .env settings (read-only)
  • System Diagnostics: Check database, models, and FAISS index health
  • Audit Log: Review recent configuration and operation changes
  • Backup/Restore: Export or import database contents as JSON
